## Frequently Asked Questions

**Can any barn be converted to commercial use?**
Most barns can be converted, but the extent of structural modification varies greatly. Some may need minor reinforcement while others require substantial reconstruction. Professional assessment determines feasibility and cost.

**What code requirements apply when converting agricultural to commercial?**
California Building Code requires structures meet occupancy-specific requirements for the new use. Assembly, restaurant, and retail occupancies each have different structural, fire safety, and accessibility requirements.

**Do agricultural conversions require seismic upgrades?**
Often yes, particularly for older structures. California Building Code triggers seismic evaluation for occupancy changes and significant renovations. The extent of upgrade depends on existing structure and new use.
