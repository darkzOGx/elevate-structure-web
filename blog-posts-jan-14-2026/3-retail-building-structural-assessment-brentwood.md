## Types of Structural Assessments

### Property Condition Assessment (PCA)

Property condition assessments include visual structural inspection, material identification, condition rating, deferred maintenance identification, and capital reserve recommendations.

### Seismic Vulnerability Assessment

Seismic assessments evaluate building classification, code compliance review, soft-story evaluation, Probable Maximum Loss (PML), and retrofit recommendations.

### Renovation Feasibility Assessment

Pre-renovation assessments determine existing structural capacity, modification potential, foundation adequacy, code upgrade triggers, and structural upgrade costs.

## The Assessment Process

### Phase 1: Document Review (1-3 days)

Review available drawings, research permit history, examine prior reports, identify building age and type.

### Phase 2: Site Inspection (1-2 days)

Visual examination of accessible areas, photographic documentation, material verification, condition notation.

### Phase 3: Analysis and Reporting (1-2 weeks)

Compile findings, rate component conditions, develop recommendations, estimate repair costs, prepare written report.
